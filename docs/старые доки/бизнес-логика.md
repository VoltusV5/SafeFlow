
1. Ветка «Подписка» (/subscribe)

    Проверка: Бот автоматически проверяет TG ID юзера на наличие использованного тест-периода.

    Если тест НЕ использовался: Автоматически включаются 3 дня триала (до 2 устройств) ➔ Сразу переход к генерации и отправке ключей.

    Если тест УЖЕ использовался: Бот предлагает выбрать платный тариф (каждый дает доступ на 3 устройства):

        Базовый: 70₽/мес или 600₽/год.

        Премиум: 129₽/мес или 999₽/год.

    Оформление: Юзер выбирает страну VPN ➔ Переходит к кнопке «Оплатить».

2. Ветка «Промокод» (/promo)

    Юзер вводит код ➔ Бот проверяет его валидность.

    Если неверный: Возврат на шаг ввода.

    Если верный: Автоматически применяется скидка (вплоть до 100%) ➔ Юзер перенаправляется на шаг «Оплата» для завершения заказа.

3. Финал (После успешной оплаты или активации теста)

    Бот автоматически генерирует 3 ключа под устройства.

    Отправляет готовые ключи пользователю в чат.



1. Ветка «Перегенерация ключей» (/regenerate)

    Цель: Дать пользователю возможность исправить работу VPN самому, если ключи вдруг перестали работать.

    Процесс (полный автомат):

        Бот удаляет старые неработающие ключи.

        Генерирует новые рабочие ключи на ноде.

        Отправляет свежие ключи пользователю в чат.

2. Ветка «Мои данные» (/my)

    Действие: Показывает пользователю информацию о его текущем профиле: активный тарифный план (Базовый/Премиум) и точную дату окончания действия подписки.

3. Ветка «Мои ключи» (/keys)

    Действие: Повторно выводит в чат текущие активные VPN-ключи пользователя (чтобы ему не приходилось пролистывать историю переписки с ботом наверх в поисках старого сообщения).




1. Ветка «Инструкции» (/help)

    Главное меню: Показывает описание всех доступных команд бота.

    Гайды по установке: При переходе в помощь с настройкой предлагает выбрать операционную систему (iOS, Android, Windows, macOS) и автоматически отправляет нужную инструкцию.

    База знаний: Содержит справочную информацию о правилах возврата средств и условиях апгрейда подписки.

2. Ветка «Возврат средств» (/refund)

    Процесс (полный автомат):

        Бот рассчитывает остаток дней до конца действия текущей подписки.

        Вычисляет пропорциональную сумму (например, если прошло 15 дней из 30, вернется 50% стоимости).

        Автоматически оформляет возврат этой суммы пользователю.

3. Ветка «Апгрейд подписки» (/upgrade)

    Процесс:

        Используется для перехода с обычной (Базовой) подписки на Премиум.

        Бот автоматически рассчитывает разницу в стоимости тарифов за те дни, что остались до конца текущей подписки.

        Выдает кнопку «Оплатить апгрейд».

        После успешной транзакции автоматически переключает пользователя на Премиум-тариф.



Сделать реферальную систему




1. Фоновые задачи (Автоматические процессы)

    Контроль подписок (по расписанию):

        Если до конца подписки осталось 3 дня ➔ Бот автоматически отправляет пользователю уведомление-напоминание.

        Если подписка закончилась ➔ Бот автоматически удаляет VPN-ключи пользователя ровно через 24 часа.

    Динамический шейпинг (на сервере):

        Если сервер перегружен ➔ Автоматически активируется лимит скорости для пользователей на «Базовом» тарифе.

        Если нагрузки нет ➔ Для всех пользователей автоматически возвращается максимальная скорость соединения.

2. Админ-панель (/admin только для админов)

    Управление промокодами: Конструктор для генерации кодов (задается процент скидки и количество возможных использований).

    Управление пользователями (Модерация):

        Кнопка ручного удаления ключей.

        Команда /ban <username/user_id> — полная блокировка пользователя в боте.

    Ручной выпуск: Кнопка «Сгенерировать ключ» для ручного создания нового конфига.

    Массовая перегенерация (на случай сбоев на нодах):

        Для одного: Кнопка «Перегенерировать @username» — обновляет и автоматически отправляет новые ключи конкретному человеку.

        Для всех: Кнопка «Перегенерировать всем» — автоматически перевыпускает ключи для всей базы пользователей и рассылает новые конфиги им в чаты.
